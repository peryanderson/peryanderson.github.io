Ruby Enumerables Cheatsheet

By Perry Anderson

Methods Function
all? [{ |obj| block } ] Checks to see if all elements of a block return true values
any? [{ |obj| block }] Checks to see if any elements of a block return true values
chunk(initial_state) { |elt, state| ... } Groups items in the block with the same succesive return values together
collect { |obj| block } Collects the results of calling the block on an object in an array
cycle(n=nil) { |obj| block } Applies the block to the object n times, if n is nil loops forever
drop(n) Drops first n elements from an object, and returns the rest.
each_cons(n) { ... } Iterates the given block for each array of consecutive elements. If no block is given, returns an enumerator.
each_entry { |obj| block } Calls block once for each element in self, passing that element as a parameter, converting multiple values from yield to an array.
each_slice(n) { ... } Iterates the given block for each slice of elements. If no block is given, returns an enumerator.
each_with_index(*args) { |obj, i| block } Calls block with two arguments, the item and its index, for each item in enum. Given arguments are passed through to each().
each_with_object(obj) { |(*args), memo_obj| ... } Iterates the given block for each element with an arbitrary object given, and returns the initially given object.
entries(*args) Returns an array containing the items in enum.
find(ifnone = nil) { |obj| block } Returns an array containing all elements of enum for which the given block returns a true value.
first(n) Returns the first element, or the first n elements, of the enumerable.
flat_map { |obj| block } Returns a new array with the concatenated results of running block once for every element in enum.
grep(pattern) { |obj| block } Returns an array of every element in enum for which pattern is equal to element. If block is included it is passed on each of these elements.
group_by { |obj| block } Returns a hash where the keys are the evaluated result from the block and the values are arrays of elements in the collection that correspond to the key.
include?(obj) Returns true if any member of enum equals obj. Equality is tested using ==.
inject(initial) { |memo, obj| block } Combines all elements of enum by applying a binary operation, specified by a block or a symbol that names a method or operator. Block and intial values are optional
lazy Returns a lazy enumerator, whose methods map/collect, flat_map/collect_concat, select/find_all, reject, grep, zip, take, #take_while, drop, and #drop_while enumerate values only on an as-needed basis. However, if a block is given to zip, values are enumerated immediately
map { |obj| block } Returns a new array with the results of running block once for every element in enum.
max { |a, b| block } Returns the object in enum with the maximum value. Block is optional way to specifiy how to organize the enum.
max_by { |obj| block } Returns the object in enum that gives the maximum value from the given block
min, min_by What they sound like, #max in reverse
member?(obj) Checks to see if any elements of a block return true values
minmax { |a, b| block }] Returns two elements array which contains the minimum and the maximum value in the enumerable
none? [{ |obj| block }] Opposite of #any
one? [{ |obj| block }] Passes each element of the collection to the given block. The method returns true if the block returns true exactly once
partition { |obj| block } Returns two arrays, the first containing the elements of enum for which the block evaluates to true, the second containing the rest
reduce(initial, sym) Combines all elements of enum by applying a binary operation, specified by a block or a symbol that names a method or operator
reject { |obj| block } Returns an array for all elements of enum for which the given block returns false
reverse_each(*args) { |item| block } Builds a temporary array and traverses that array in reverse order
select { |obj| block } Returns an array containing all elements of enum for which the given block returns a true value.
slice_before(pattern) Creates an enumerator for each chunked elements. The beginnings of chunks are defined by pattern and the block
sort { |a, b| block } Returns an array containing the items in enum sorted
sort_by { |obj| block } Sorts enum using a set of keys generated by mapping the values in enum through the given block
take(n) Returns first n elements from enum
to_a(*args) Returns an array containing the items in enum.
to_h(*args)] Returns the result of interpreting enum as a list of [key, value] pairs.
zip(arg, ...) { |arr| block } Cakes one element from enum and merges corresponding elements from each args. This generates a sequence of n-element arrays, where n is one more than the count of arguments
